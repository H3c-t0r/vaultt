# `/identity/mfa`

## Supported MFA types.

- [TOTP](/api/identity/mfa/totp)

- [Okta](/api/identity/mfa/okta)

- [Duo](/api/identity/mfa/duo)

- [PingID](/api/identity/mfa/pingid)

## Other

- [Login Enforcement](/api/identity/mfa/login-enforcement)

Note that while the above endpoints are available in both the open source and Enterprise versions of Vault,
they are namespace aware. MFA methods created in one namespace are separate from other namespaces. In the open
source version of Vault, everything operates in the root namespace.
